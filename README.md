  # ⚽ Football Match Converter

Tool sederhana untuk mengubah data pertandingan menjadi HTML.

## 🚀 Cara Pakai

1. Copy data pertandingan
2. Paste ke textarea
3. Klik tombol **Convert**
4. Hasil HTML muncul + preview

## 🌐 Demo Lokal

Buka `index.html` di browser.

## 📦 Fitur

- Auto grouping liga
- Parse tanggal & jam
- Hapus ranking [xx]
- Generate HTML otomatis
